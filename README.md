🚗 Car Price Prediction Project

📌 About Project
This is a complete Machine Learning project that predicts car prices based on different features like brand, mileage, engine size, fuel type, transmission, and number of owners.
The project also includes data cleaning, feature selection, model training, and evaluation.
⚙️ What This Project Does
Synthetic (real-world style) car dataset generate karta hai
Data cleaning perform karta hai
Missing values, duplicates aur outliers handle karta hai
Feature engineering (car age create karta hai)
Feature selection analysis karta hai
Regression models train karta hai
Model evaluation (MAE, RMSE, R²) calculate karta hai

🧠 Machine Learning Models Used
Linear Regression
Random Forest Regressor

📊 Evaluation Metrics
Model performance check ki jati hai using:
MAE (Mean Absolute Error)
RMSE (Root Mean Squared Error)
R² Score
📈 Results
Random Forest generally better performance deta hai
Model car price ko fairly accurate predict karta hai
Best model is selected based on lowest RMSE
